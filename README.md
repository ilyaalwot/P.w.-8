# Практична робота №7
## Колекції в Java. Створення власної примітивної колекції типу Vector

У роботі реалізовано власну колекцію типу Vector для зберігання об’єктів Book без використання Java Collection Framework та generics.

## Реалізовані методи:
- add(Book book) — додавання в кінець
- addFirst(Book book) — додавання на початок
- add(int index, Book book) — додавання за індексом
- get(int index) — отримання елемента за індексом
- remove(int index) — видалення елемента
- clear() — очищення колекції
- size() — кількість елементів
- capacity() — поточна місткість масиву
