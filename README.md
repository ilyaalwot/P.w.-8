# Практична робота №8
## Використання виключень користувача

У роботі реалізовано власну примітивну колекцію типу Vector для зберігання об’єктів Book.
На основі практичної роботи №7 було додано користувацькі виключення для обробки помилкових ситуацій.

## Реалізовані виключення:
- InvalidCapacityException
- EmptyVectorException
- InvalidIndexException
- NullBookException
- VectorStateException

## Реалізовані можливості:
- додавання елементів у кінець, початок та середину;
- отримання елемента за індексом;
- видалення елемента;
- очищення колекції;
- перевірка size та capacity;
- обробка помилок через власні виключення.
